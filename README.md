# Portfolio — Matéo Fourquet

Site statique, aucun build requis.

## Contenu
- index.html — le site
- support.js — runtime (requis, même dossier que index.html)
- uploads/CV_Mateo_Fourquet_v4.pdf — le CV lié au bouton "Résumé"

## Déployer sur GitHub Pages
1. Crée un repo public nommé mateofrqt.github.io
2. Uploade le CONTENU de ce dossier à la racine du repo
3. Settings → Pages → Source: "Deploy from a branch" → main → /(root)
4. Le site est en ligne sur https://mateofrqt.github.io (compte ~1 min)

## Alternatives
- Netlify / Cloudflare Pages : glisse ce dossier dans leur interface, c'est tout.

## Mettre à jour
Remplace les fichiers par leurs nouvelles versions (mêmes noms) et push.
